================================================================================
                      HOW TO USE THE APP
================================================================================

CREATE A TODO:
   1. Click "+" button in header
   2. Enter title (required)
   3. Add description (optional)
   4. Select due date (optional)
   5. Click "Create"

EDIT A TODO:
   1. Click on any todo card
   2. Modal opens with current values
   3. Make changes
   4. Click "Update"

DELETE A TODO:
   - Desktop: Hover and click delete icon
   - Mobile: Swipe left to reveal delete

MARK COMPLETE:
   - Click checkbox on left side of todo

SEARCH:
   - Type in search bar
   - Filters by title and description

FILTER:
   - Click tabs: All, Active, Completed, Due Soon

REORDER:
   - Hover to see drag handle
   - Drag to new position
   - Release to save

SWITCH THEME:
   - Click sun/moon icon in header
   - All colors smoothly transition
